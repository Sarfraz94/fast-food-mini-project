# Fast Food Cards UI

A modern Fast Food User Interface built using HTML5 and CSS3. This project showcases a food ordering layout with a navigation bar, image slider, and product cards. The main focus of the project is to practice frontend layout design, card-based components, positioning, and styling techniques.

## Live Demo

🔗 https://sarfraz94.github.io/fast-food-cards-ui-html-css/

## Overview

The interface features a fixed navigation bar, search section, image slider, and a collection of food cards displaying product images, prices, ratings, and favorite icons. The project was developed to strengthen core frontend development skills using only HTML and CSS.
